## Deskripsi Karya
EduPath adalah platform web pendidikan yang dirancang dengan mengimplementasikan tema INNOVATE untuk membantu navigasi pembelajaran digital. Website ini menggunakan layout Bento Grid untuk aspek estetika dan fungsionalitas optimal, serta dilengkapi dengan fitur kuis adaptif untuk validasi pemahaman pengguna.

### Implementasi Tema INNOVATE:

1. **Impel Novelty**: Mengganti daftar kursus konvensional dengan Visual Learning Tree yang interaktif
2. **Navigate**: Fitur roadmap belajar yang membimbing pengguna dari level dasar hingga mahir
3. **Optimize**: Tata letak Bento Grid untuk mengoptimalkan ruang dan keterbacaan informasi
4. **Validate**: Modul kuis adaptif di setiap akhir materi untuk memvalidasi pemahaman
5. **Advance**: Mendorong peningkatan keahlian digital masyarakat melalui akses pengetahuan terorganisir

### Fitur Utama:
- Visual Learning Tree dengan sistem unlock progresif
- Sistem kuis adaptif untuk validasi pemahaman
- Desain Bento Grid yang modern dan responsif
- Navigasi yang intuitif dengan Bootstrap 5
- Animasi smooth dengan AOS library
- Sistem level (Beginner, Intermediate, Advanced)
- Progress tracking per modul

### Teknologi yang Digunakan:
- **HTML5** - Struktur halaman
- **CSS3** - Styling custom dengan variabel CSS
- **Bootstrap 5** - Framework CSS utama
- **Bootstrap Icons** - Icon set modern
- **AOS Library** - Animasi scroll
- **Google Fonts** - Plus Jakarta Sans
- **JavaScript Vanilla** - Interaktivitas tanpa framework

## Struktur Halaman
1. **Home** (index.html): Hero section dengan ilustrasi modern, implementasi INNOVATE, fitur unggulan
2. **About** (pages/about.html): Penjelasan visi, misi, implementasi INNOVATE, timeline kegiatan
3. **Roadmap** (pages/roadmap.html): Visual Learning Tree dengan sistem unlock dan kuis adaptif
4. **Contact** (pages/contact.html): Formulir kontak, informasi tim, dan detail pembayaran
